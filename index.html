<p></p> The paragraph tags have been used at the beginning and the end of this paragraph and each following paragraph in this section. This applies a default non-heading font size and generally ensures at least two line breaks at the beginning and end of the paragraph without creating additional breaks where they aren't necessary due to adjacent paragraph or heading tags.

<br> Break
tags have been used after the first word of this sentence and before the
last.

<ul></ul> Unordered list tags have been used in the All Elements section below to present the links as an unordered list.

<a href=""></a> or <a href=""></a> Anchor tags have been used in the All Elements section below to add hyperlinks to the URL text. The <a href=""></a> tag is used to turn any text into a hyperlink and underline it, as in the first two links below. The <a></a> tag by itself will turn the URL itself into a hyperlink with no underline, as in the second two links below.

<span></span> Span tags have been used between the brackets and the rest of the tag in the tags that are visible on the webpage above this this paragraph, in order to render them as text rather than being computed by the browser as tags. However, a simpler way to do this is by using "& lt" and "& gt" for the less than (<) and greater than (>) signs respectively, which has been done in this paragraph.
